# Changelog

All notable changes to the Anti Enderman Grief scarpet app will be documented in this file.

## [v1.1.0]
* **Feature Update:** Modified the `enderman_holdable` tag injection to allow Endermen to pick up `minecraft:melon` and `minecraft:pumpkin` blocks.
* **Documentation:** Updated the in-game `info` command to reflect the new block exceptions.

## [v1.0.0]
* **Initial Release:** Replicated the Vanilla Tweaks Anti Enderman Grief datapack logic in Scarpet.
* **Feature:** Implemented virtual datapack injection using `create_datapack()` to override the `minecraft:enderman_holdable` block tag efficiently.
* **Command:** Added the `info` command to display app metadata and credits in-game.
